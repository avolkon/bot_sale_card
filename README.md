\# Бот-агент для создания карточек продуктов



\## Описание

Бот для Telegram, который парсит страницу продукта по URL и генерирует продающую карточку по шаблону с помощью DeepSeek API.



\## Функции

\- Парсинг HTML страницы с группировкой по блокам

\- Генерация карточки по шаблону через LLM

\- Постобработка текста маркетологом

\- Согласование с клиентом (кнопки "Всё хорошо" / "Внести правки")

\- Сохранение финальной карточки в TXT файл
